Promise It Won't Hurt
10/19/2015

Node.js achieves its performance through the use of asynchronous I/O. Although this is terrific for performance, asynchronous code can be very difficult for programmers to manage. As a result, developers often spend too much time debugging obscure problems and writing lots of code.

JavaScript to the rescue! JavaScript by itself is a great language to use when dealing with asynchronicity. Promises make that story even better by making asynchronous code composable. Work through this workshopper to find out how.
